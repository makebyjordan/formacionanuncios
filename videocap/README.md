# Videocapturas (Videocap)

Sube tus grabaciones de pantalla del Mac a esta carpeta. 

### Instrucciones:
1. Graba tu pantalla en tu Mac.
2. Si es necesario, comprime o reduce el tamaño del vídeo.
3. Copia el archivo de vídeo (ej. `.mp4`, `.mov`, `.webm`) en esta carpeta `videocap/`.
4. Escribe **ANALIZA** en la conversación. El script copiará el vídeo automáticamente al proyecto web, actualizará la base de datos y lo desplegará.
